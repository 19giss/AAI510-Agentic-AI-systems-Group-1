# Project Scope

## MVP Scope

The first version of VibeML will focus on creating a playlist recommendation using the Kaggle Spotify datasets and a simulated or available Spotify user profile.

The MVP should include:

- Data loading and cleaning
- Song search based on audio features
- Vibe mapping from user request to music features
- Tester song recommendations
- Direct user feedback on tester songs
- Feedback refinement based on accepted or rejected songs
- Final playlist output
- Evaluation traces that include user input, tool calls, output, user feedback, and human review

## Stretch Goals

These may be added if time allows:

- Live Spotify API authentication
- Pulling real liked songs or playlists from a team member's Spotify account
- Saving the final playlist directly to Spotify
- Adding a simple memory layer for user preferences
- Using stored user preferences to improve future playlist recommendations
- More advanced playlist ordering
